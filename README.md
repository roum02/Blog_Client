# Blog_Client

기술 스택

- Frontend: Next.js 15 (App Router), React Query, Froala Editor
- Backend: NestJS (Docker EC2 배포), RDS (PostgreSQL)
- Editor: react-froala-wysiwyg
- Storage: EC2 내부 디렉토리 (/uploads/images) 사용
